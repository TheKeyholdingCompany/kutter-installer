# Kutter CLI Installer

Install the `kutter` CLI for generating Kotlin microservices from the Kutter template.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/TheKeyholdingCompany/kutter-installer/main/install.sh | bash
````

## Usage

```bash
kutter new my-little-project
```

## Notes

* Requires `git`, Homebrew, and SSH access to the private Kutter template repository
* The installer is public; the template repository is private
